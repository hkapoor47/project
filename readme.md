# Node Authentication API

A simple authentication API built with Node.js, Express, MongoDB, JWT, and bcrypt.

## Features

- User Registration
- User Login
- Password Hashing
- JWT Authentication
- Protected Routes

## Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- bcrypt
- JSON Web Token (JWT)

## Installation

```bash
npm install
```

Create a `.env` file:

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```

Start the server:

```bash
node server.js
```
